# Environment variables the linodemcp binaries read, one per line, sorted.
# This is the COMPLETE env-read surface for every registered language:
# scripts/verify_env_parity.py extracts the env-lookup literals from each
# implementation's source and fails unless every language reads exactly this
# set. Adding a line here means implementing the same read in every language
# in the same change; removing a read means removing it everywhere and
# deleting the line.
#
# Observability (metrics, tracing, health) has no env overrides on purpose:
# those knobs live in the config file only, so one config parses to one
# behavior in every language and in every shell.
